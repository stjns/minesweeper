Right click/long press to set flags, left click/short press to reveal cells.

Left click/short press on a "numbered" cell to chord (reveal all unrevealed cells it touches, assuming the number of nearby flags matches the number on the cell).

Right click/long press on a "numbered" cell to flag all cells it touches if the number of unrevealed cells it touches matches the number on the cell.
